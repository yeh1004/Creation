# 📡 實境解謎專案：《代號：特普任務》工作人員指引手冊
Welcome to the official Master Control and Facilitator Guide for **《代號：特普任務》 (Operation: TRUMP)**! 

本專案是一款結合**校園實地防災探索**、**急救包紮實作**與**媒體識讀/防詐防霸凌**的高質感網頁戰術平板實境解謎遊戲。本手冊旨在為現場主控工作人員（小幫手、評審、引導員）提供最完整的解謎邏輯、標準答案，以及一套**「三階段（輕/中/強）漸進式提示指引」**，讓工作人員在不直接劇透的前提下，有效引導卡關小隊順暢通關，維持極致的沉浸式遊戲體驗。

---

## 🧭 快速啟動與系統概述

### 1. 執行方式
* 本專案為單一 HTML 檔案架構，雙擊 `代號：特普任務.html` 即可於瀏覽器（推薦使用 Chrome 或 Safari 手機版）中直接執行。
* **支援設備**：極致適應行動端（RWD 響應式佈局），玩家與工作人員使用手機或平板即可流暢體驗。

### 2. 工作人員中控平台（後台）
* **登入管道**：於首頁點擊下方「工作人員登入」，並輸入安全密鑰 **`286`** 即可進入中控平台。
* **後台功能**：
  * **各關答案參考**：即時查閱各關卡答案。
  * **即時排名監控**：實時掌握所有註冊小隊的**通關進度**、**第一通關成績**（鎖定首次破關記錄，二遊不覆蓋）、**累計失敗次數**、**闖關次數**與**動態榮譽稱號**。
  * **測試工具**：提供快速測試面板，可模擬玩家跳轉關卡或重置狀態。

### 📦 任務二：物資精算師 (背包限重優化與密碼鎖)
* **關卡主題**：避難物資的實用性評估與預算控制。
* **解謎工具**：防災包模擬選擇介面、生存點數雷達圖、解密鍵盤。

#### 📊 關卡解謎與思考邏輯
* **規格限制**：
  1. 預算金額：**`<= 1000 元`**
  2. 背包總重：**`<= 3500g`**
  3. 生存價值分數：**`>= 80 分`**
* **核心必選物資**（沒選將被退回）：
  * 💧 **瓶裝水** (Survival: +25, $20, 650g) — 維持生命絕對必需。
  * 📢 **求生哨** (Survival: +8, $25, 15g) — 瓦礫堆下求救必備。
  * 🍪 **高熱量食物** (壓縮餅乾/巧克力/能量棒 三者任選其一) — 補充體力。
* **特殊彩蛋（皮卡丘收服降重機制）**：
  * 若玩家在背包中同時選取了 **`神奇寶貝球`🔴** 與 **`皮卡丘`⚡**，將會觸發超自然羈絆彩蛋，皮卡丘的 `6000g` 體重會**歸零**！此彩蛋是讓玩家帶走皮卡丘且不超重的唯一方法。
* **密碼鎖終端密碼**：**`0 - 3 - 0`** (來自於防災包通過篩選後，背包明細所給予的終端綠色協議提示)。
* 🔊 **玩家成功提示語**：`"解鎖取得「主機室通行卡」與物資箱位置座標(0,0)"`

#### 🗺️ 關卡間地點安全驗證（攔截問答）
* **問題**：根據已知故事情節填入下一任務的場所，此處在校園何處？
* **正確答案**：**`健康中心`**
* 🔊 **驗證成功提示**：`"🔑 地點安全驗證成功！前往下一關卡的傳輸協議已解鎖。"`

#### 💡 工作人員三階段提示卡
* 🟢 **輕度提示 (淺 - 引導思考)**：
  > 「水和求救工具是生命底線，你的背包裡有帶水和能發出聲音的東西嗎？泡麵沒有熱水能吃嗎？」
  > 「生存價值高但不實用的重物（如啞鈴、原文書、VR眼鏡）會拖垮體力，應該果斷捨棄！」
* 🟡 **中度提示 (中 - 縮小範圍)**：

* 🔴 **強度提示 (高 - 直指關鍵)**：
  > 「選擇瓶裝水、求生哨、壓縮餅乾，再搭配其他高分輕量物品（如手電筒、雨衣、急救包等）使生存分數超過 80。成功後會解鎖密碼程序，輸入密碼 `0-3-0` 即可通關。接下來的關卡拦截問答輸入『健康中心』！」

### 🛡️ 任務四：真相的迷霧 (媒體識讀、反匿名霸凌與數位防詐)
* **關卡主題**：假新聞過濾、影像真偽鑑定與官方安全防域確認。
* **解謎工具**：情報卡片排查介面、影像鑑定掃描板、教育部官方網址比對。

#### 📊 關卡解謎與思考邏輯

##### 🕵️ 階段 1：火警情報解析
* **謎題機制**：玩家需過濾匿名論壇、個人社交媒體及內容農場的假新聞。
* **排查規則**：排除匿名論壇(9), IG限動(5), 內容農場(8)等假新聞，留下其餘 3 張【真實情報】，將其角落數字**由小到大**排列。
* **真實情報數字**：`0`、`3`、`7`。
* **標準答案**：**`037`**
* 🔊 **成功提示語**：`"火警情報 analysis 成功！淘汰了假新聞，解鎖鑑識平板！"`

##### 🔍 階段 2：影像圖層鑑識
* **謎題機制**：點擊照片檢視，將半透明掃描板疊加在「無破綻的真實現場照片 C」上。
* **解密線索**：透過紅框掃描區域，顯現出隱藏的 4 位數密碼。
* **標準答案**：**`2048`**
* 🔊 **成功提示語**：`"影像鑑識成功！鎖定真實場景，開啟最後的官方防線網域！"`

##### 🌐 階段 3：官方安全網域驗證
* **謎題機制**：防範數位詐騙釣魚網址，點選正確的教育部校園安全官方防線連結。
* **網址比對**：
  * A. `www.edu.tw.campus-security.login-auth.com` (❌ 假網域釣魚)
  * B. `www.edutw-campus-safety.org` (❌ 拼寫偽造)
  * C. `www.edu.tw/campus-safety` (✅ 正確教育部二級目錄官網)
* **標準答案**：**選項 C**
* 🔊 **成功提示語**：`"官方防線驗證成功，通關成功！"`

* 🗺️ **最終解鎖定位提示**：**`座標 (-2, 6)`** （對應校園實地場所：**維新館**）。
